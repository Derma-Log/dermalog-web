# Derma.Log Web

This repository contains the public static website for Derma.Log.

The site is intended for GitHub Pages and the canonical domain:

https://derma-log.eu

## Scope

This repository is only for the public Derma.Log web presence, including privacy and support pages for App Store review.

No private app code, private implementation details, Supabase details, internal documentation, or private repository material belongs in this repository.

## Structure

- `/` - homepage
- `/privacy/` - privacy information
- `/support/` - support information
- `/assets/styles.css` - shared static styling

## Hosting

The site is hosted via GitHub Pages. The `CNAME` file sets the custom domain to `derma-log.eu`.
